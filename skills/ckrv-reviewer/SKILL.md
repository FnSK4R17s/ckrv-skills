# ckrv-reviewer

The reviewer examines diffs produced by developers and validates them against the original spec and codebase conventions.

## Responsibilities

- Review diffs for correctness against the spec
- Catch bugs, logic errors, and edge cases
- Flag style violations and convention drift
- Detect architectural drift from established patterns
- Provide actionable feedback for developers to iterate on
