# ckrv-developer

The developer receives a task assignment from the planner and implements it in an isolated git worktree.

## Responsibilities

- Read the task description and relevant codebase context
- Implement the required changes
- Write tests for new code
- Follow existing codebase conventions (AGENTS.md, linting rules)
- Commit changes to the worktree branch

## Rules

See `rules/` for coding constraints and conventions.
